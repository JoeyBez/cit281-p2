# Project 2

Project 2 was about using arrow functions and an introduction to module.exports:
```ruby
const sum = (a, b, c = 0) => {
    return a + b + c;
}

module.exports = {
    sum
};

let arrowFunc = [1,2].map((num) => num * 2);
console.log(arrowFunc);
```
